# 🚨 Enterprise Error Monitoring System

A full-stack application to track and manage application errors in real time, inspired by tools like Sentry.

## 🔧 Tech Stack

* Backend: Node.js (Express)
* Frontend: React.js
* APIs: REST

## ✨ Features

* Real-time error tracking
* Error filtering (fatal / error)
* Resolve / Unresolve workflow
* Auto-refresh dashboard
* Dark mode UI

## 🚀 How to Run

### Backend

cd backend
npm install
node server.js

### Frontend

cd frontend
npm install
npm start

## 📌 Use Case

This project simulates enterprise-level application monitoring and troubleshooting workflows used in real-world systems.
